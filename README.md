
# WE-BJUT

## 项目背景

在校内学习和生活遇到的主要问题集中在获取教务信息难（连校网-打开教务系统-查询）、公众号消息混乱（资讯、服务和干货混淆在一起）、每年产生大量重复内容、零碎性消息获取困难等。

## 解决方案

**基础服务**：查课表、查空教室、查成绩等。

**攻略**：校园信息分流、精选优质文章、沉淀学长学姐经验等。

**智能机器人**：回答固定消息，为辅导员分忧。
